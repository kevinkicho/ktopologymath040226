Decoherence

Quantum states are fragile. Interaction with the environment causes two types of errors:

  • Amplitude damping (T₁): Energy loss — qubit decays from |1⟩ to |0⟩
  • Dephasing (T₂): Phase randomization — off-diagonal elements vanish

Lindblad Master Equation

$$\frac{d\rho}{dt}=\sum_k\!\left(L_k\rho L_k^\dagger-\tfrac{1}{2}\{L_k^\dagger L_k,\rho\}\right)$$

Kraus Operators — Amplitude Damping

$$K_0=\begin{pmatrix}1&0\\0&\sqrt{1-\gamma}\end{pmatrix},\quad K_1=\begin{pmatrix}0&\sqrt{\gamma}\\0&0\end{pmatrix}$$

Dephasing (T₂)

$$\rho_{01}(t)=\rho_{01}(0)\,e^{-t/T_2}$$

Off-diagonal elements (coherences) decay exponentially. The diagonal (populations) are preserved.

No-Cloning Theorem

An unknown quantum state cannot be copied: $\nexists\, U$ such that $U|\psi\rangle|0\rangle=|\psi\rangle|\psi\rangle$ for all $|\psi\rangle$. Classical error correction (redundant copies) is forbidden! QEC must instead encode into entangled states and detect errors without measuring the logical qubit.

Bloch Sphere

$$\rho=\frac{1}{2}(I+\vec{r}\cdot\vec{\sigma}),\quad |\vec{r}|\le 1$$

Pure states: $|\vec{r}|=1$ (surface). Mixed states: interior. Decoherence shrinks the Bloch vector toward the center or toward the north pole.

Bit-Flip Code [[3,1,1]]

$$|\bar{0}\rangle=|000\rangle,\quad|\bar{1}\rangle=|111\rangle$$

Encodes 1 logical qubit in 3 physical qubits. Corrects a single bit-flip (X error).

Syndrome Measurement

$$S_1=Z_1Z_2,\quad S_2=Z_2Z_3$$

Syndrome eigenvalues ±1 identify the error without measuring the logical qubit:

  • $S_1=-1,\,S_2=+1$ → error on Q₁
  • $S_1=-1,\,S_2=-1$ → error on Q₂
  • $S_1=+1,\,S_2=-1$ → error on Q₃
  • $S_1=+1,\,S_2=+1$ → no error

Phase-Flip Code

$$|\bar{0}\rangle=|{+}{+}{+}\rangle,\quad|\bar{1}\rangle=|{-}{-}{-}\rangle$$

Apply Hadamard to each qubit, then use bit-flip syndrome in the X basis via $S_1=X_1X_2,\,S_2=X_2X_3$.

No-Cloning Relevance

These codes do not copy the logical state — they entangle it into a joint state. The syndrome only reveals which physical error occurred, never the logical information.

Limitations

The bit-flip code corrects X errors but not Z errors, and vice versa. Shor's 9-qubit code combines both.

Stabilizer Group

A stabilizer code $\mathcal{C}(S)$ is the simultaneous +1 eigenspace of an Abelian group $S$ of Pauli operators:

$$\mathcal{C}(S)=\{|\psi\rangle : g|\psi\rangle=|\psi\rangle\;\forall g\in S\}$$

[[n, k, d]] Parameters

  • $n$ = physical qubits
  • $k = n - m$ = logical qubits (m = independent generators)
  • $d$ = minimum distance (smallest weight undetectable error)

Pauli Group

$$\mathcal{P}_1=\{\pm I,\pm iI,\pm X,\pm iX,\pm Y,\pm iY,\pm Z,\pm iZ\}$$

Products of Paulis pick up phases ±1, ±i. The table shows products ignoring global phase.

Logical Operators

Logical $\bar{X}$ and $\bar{Z}$ commute with all stabilizers but are not in $S$ itself. They act on the logical qubit without destroying the code space.

$$[\bar{X},g]=[\bar{Z},g]=0\;\forall g\in S,\quad \{\bar{X},\bar{Z}\}=0$$

Error Detection

An error $E$ is detectable iff $\exists g\in S$ such that $\{E,g\}\ne 0$ (anti-commutes). The syndrome is the pattern of anti-commuting generators.

Shor/Steane Connection

The Steane [[7,1,3]] code has 6 generators (3 X-type, 3 Z-type from the [7,4,3] Hamming code), yielding $k=7-6=1$ logical qubit with distance $d=3$.

Shor Code [[9,1,3]]

Shor's 9-qubit code protects against any single-qubit error (X, Y, Z, or their combination). It works by concatenating the bit-flip code inside the phase-flip code:

$$|\bar{0}\rangle=\frac{1}{2\sqrt{2}}(|000\rangle+|111\rangle)^{\otimes 3}$$
$$|\bar{1}\rangle=\frac{1}{2\sqrt{2}}(|000\rangle-|111\rangle)^{\otimes 3}$$

Steane Code [[7,1,3]]

Encodes 1 logical qubit in 7, with distance $d=3$. Based on the classical [7,4,3] Hamming code via the CSS construction:

$$\mathcal{C}_{CSS}(C_1,C_2):\; C_2^\perp\subseteq C_1$$

X-type stabilizers from $H^\perp$ parity checks, Z-type from $H$ parity checks.

Transversal Gates

In both codes, logical CNOT is implemented by applying physical CNOT transversally (qubit $i$ to qubit $i$). This is fault-tolerant — errors do not spread.

$$\overline{CNOT}=CNOT_1\otimes CNOT_2\otimes\cdots\otimes CNOT_n$$

Logical Error Rate

$$p_L\sim\binom{d}{(d+1)/2}p^{(d+1)/2}\approx p^2\text{ for }d=3$$

For physical error rate $p \ll p_{th}$, logical errors are exponentially suppressed in the code distance $d$.

CSS Construction

Any pair of classical codes $C_1\supseteq C_2$ with $C_2^\perp\subseteq C_1$ yields a quantum code. This connects quantum error correction to classical coding theory.

Surface Code

The leading candidate for fault-tolerant quantum computing. A 2D lattice of data qubits with X-type (face) and Z-type (vertex) stabilizers.

Stabilizers

$$A_v=\prod_{e\ni v}X_e \quad\text{(vertex/Z-syndrome)}$$
$$B_p=\prod_{e\in p}Z_e \quad\text{(plaquette/X-syndrome)}$$

X errors create pairs of Z-syndrome defects (plaquette violations). Z errors create pairs of X-syndrome defects (vertex violations).

Threshold

$$p_{th}\approx 1\%$$

Among the highest thresholds of any code. Below this, increasing $d$ suppresses logical errors exponentially.

Logical Error Rate

$$p_L\sim\left(\frac{p}{p_{th}}\right)^{\lceil(d+1)/2\rceil}$$

For $p=0.1\%$, $d=5$: $p_L\approx 10^{-9}$.

Logical Operators

Logical $\bar{X}$: horizontal chain of X operators across the lattice. Logical $\bar{Z}$: vertical chain of Z operators. These are topologically protected — only chains spanning the full lattice cause logical errors.

Decoder

Minimum-weight perfect matching (MWPM) pairs syndrome defects optimally. Near the threshold, machine-learning decoders (neural networks) outperform MWPM.

Threshold Theorem

If physical error rate $p < p_{th}$, then logical error rate can be made arbitrarily small by increasing code size $d$:

$$p_L\le\left(\frac{p}{p_{th}}\right)^{\lceil(d+1)/2\rceil}$$

This is the central result that makes large-scale quantum computing theoretically possible.

Fault-Tolerant Circuit Design

Key principles:

  • No single fault causes two errors: ancilla preparation and measurement must be fault-tolerant
  • Transversal gates: apply gate qubit-by-qubit; errors cannot propagate within a codeblock
  • Cat states: for non-transversal syndrome extraction

Magic State Distillation

Universal FT quantum computing requires the T gate ($\pi/8$ rotation). T is not transversal in most codes. Instead:

$$|T\rangle=T|{+}\rangle=\frac{|0\rangle+e^{i\pi/4}|1\rangle}{\sqrt{2}}$$

Distill many noisy $|T\rangle$ states into fewer high-fidelity ones using Clifford operations and post-selection. Overhead: ~1000 physical qubits per logical T gate.

Concatenation

Concatenating the Steane code $k$ times uses $7^k$ physical qubits per logical qubit and achieves:

$$p_L\le p_{th}\left(\frac{p}{p_{th}}\right)^{2^k}$$

Resource Estimates

Factoring a 2048-bit number (breaking RSA) requires ~$4\times 10^6$ physical qubits at $p=10^{-3}$, running for ~8 hours of circuit time.

Toric Code

Kitaev's toric code lives on a torus (periodic boundary conditions). It has two types of anyonic excitations from stabilizer violations:

  • e-anyons: vertex ($A_v$) violations — from Z errors
  • m-anyons: plaquette ($B_p$) violations — from X errors

Anyon Statistics

$$e\times e=1,\quad m\times m=1,\quad e\times m=\epsilon$$

e and m are Abelian anyons. Braiding an e around an m acquires a phase of $-1$ (mutual statistics).

$$\theta_{em}=e^{i\pi}=-1$$

Topological Protection

Logical errors require creating an anyon pair, moving one across a non-contractible cycle of the torus, and annihilating. This is exponentially unlikely at low temperature.

Fibonacci Anyons

Non-Abelian anyons with fusion rule:

$$\tau\times\tau=1+\tau$$

Quantum dimension $d_\tau=\varphi=(1+\sqrt{5})/2$ (golden ratio). A system of $n$ $\tau$-anyons has Hilbert space growing as $\varphi^n$.

Braid Matrix (Fibonacci)

$$R=\begin{pmatrix}e^{-4\pi i/5}&0\\0&e^{3\pi i/5}\end{pmatrix}$$
$$F=\begin{pmatrix}\varphi^{-1}&\varphi^{-1/2}\\\varphi^{-1/2}&-\varphi^{-1}\end{pmatrix}$$

Fibonacci braids are universal for quantum computation — any unitary can be approximated to arbitrary precision.

Physical Realizations

Fractional quantum Hall ν=5/2 state may host Ising anyons. Microsoft pursues Majorana-based topological qubits (non-Abelian anyons) for inherently protected quantum computing.

Google Willow (2024)

105 superconducting qubits. First demonstration of below-threshold error rates: logical error rate decreases as code size $d$ increases from 3→5→7. Sub-threshold at $d=5$ with surface code.

IBM Roadmap

IBM targets 1000+ physical qubits per logical qubit with error rates reaching $10^{-3}$ by 2025–2026. Heron processor: 133 qubits, 2-qubit gate fidelity ~99.5%.

Overhead Reality

At current error rates (~$10^{-3}$), fault-tolerant algorithms require:

  • ~1,000 physical qubits per logical qubit
  • ~$10^{10}$ operations for RSA-2048
  • ~$4\times10^6$ total physical qubits

NISQ vs FT Era

NISQ (now): 50–1000 noisy qubits, no error correction. Variational algorithms (VQE, QAOA) attempt useful computation despite noise.

Early FT (2026–2030): First logical qubits demonstrably better than physical. Small error-corrected algorithms possible.

Full FT (2030+): Millions of physical qubits. RSA-relevant cryptanalysis, drug discovery, climate modeling.

Platforms Compared

  • Superconducting: Fast gates (~50 ns), but short $T_1\sim100\,\mu$s, cryogenic
  • Trapped ion: Long coherence ($T_2\sim$minutes), slower gates (~1 ms), optical connections
  • Photonic: Room temperature, but low interaction rates, measurement-based
  • Neutral atom: 2D arrays, fast reconfiguration, growing rapidly
