C∞ vs Cω
The bump function is infinitely differentiable everywhere yet its Taylor series at x=0 is identically zero — it is not analytic. This is impossible for complex functions: holomorphic ⇒ analytic.

C∞ partitions of unity patch together local data in differential geometry. The bump function is the prototype. Such constructions fail in complex analysis — forcing global rigidity.

Geometric Series
$\sum_{n=0}^{\infty}z^n = \dfrac{1}{1-z}$, $\quad|z|<1$
But $\frac{1}{1-z}$ exists for all $z\ne1$. Analytic continuation extends the domain.
Riemann ζ Function
$\zeta(s)=\sum_{n=1}^{\infty}n^{-s}$, $\quad\mathrm{Re}(s)>1$
Continued to $\mathbb{C}\setminus\{1\}$ via functional equation:
$\zeta(s)=2^s\pi^{s-1}\sin\!\tfrac{\pi s}{2}\,\Gamma(1{-}s)\,\zeta(1{-}s)$

Monodromy Theorem
If continuation along every path in a simply-connected domain $D$ gives single-valued results, then $f$ extends to a global holomorphic function on $D$.

Saddle Point
Step through steepest-descent approximations.
Saddle Point Formula
$$I(N)=\int_\gamma e^{N f(z)}\,dz \;\sim\; e^{N f(z_0)}\sqrt{\frac{2\pi}{N|f''(z_0)|}}$$ as $N\to\infty$, where $z_0$ is the saddle point.
Stirling's Approximation
$$N!\approx\sqrt{2\pi N}\left(\frac{N}{e}\right)^N$$ Exact via saddle of $\int_0^\infty t^N e^{-t}\,dt=\Gamma(N{+}1)$.
Steepest Descent
Deform contour through saddle point $z_0$ along the direction where $\mathrm{Re}[f(z)]$ decreases fastest: $\arg(dz)=-\tfrac{1}{2}\arg f''(z_0)$.

Kramers-Kronig Relations
$$\varepsilon_1(\omega)=1+\frac{2}{\pi}\mathcal{P}\!\int_0^\infty\!\frac{\omega'\varepsilon_2(\omega')}{\omega'^2-\omega^2}d\omega'$$ $$\varepsilon_2(\omega)=-\frac{2\omega}{\pi}\mathcal{P}\!\int_0^\infty\!\frac{\varepsilon_1(\omega')-1}{\omega'^2-\omega^2}d\omega'$$
$\mathcal{P}$ = Cauchy principal value. Follows from analyticity of $\varepsilon(\omega)$ in upper half-plane.

• Branch cuts (multi-particle thresholds)
