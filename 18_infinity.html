Proof sketch:
Suppose $f:\mathbb{N}\to[0,1]$ surjective. Write each $f(n)$ in binary. Define $d_n = 1-\text{bit}_n(f(n))$. Then the number $d = 0.d_1d_2d_3\ldots$ satisfies $d \neq f(n)$ for all $n$ (differs at $n$th digit). Contradiction.

Same proof shows:
$|\mathcal{P}(S)| > |S|$ for any set $S$.

Cantor (1891):
No list of reals in [0,1] can be complete. The diagonal number $d$ (changing each digit) is missing.
"No one shall expel us from the paradise that Cantor has created." — Hilbert
Diagonal number d will appear here.
Step 1: List all reals
$f(1)=0.d_{1,1}d_{1,2}d_{1,3}\ldots$
$f(2)=0.d_{2,1}d_{2,2}d_{2,3}\ldots$
$\vdots$
Step 2: Define diagonal:
$d_n \neq d_{n,n}$
Step 3: $d=0.d_1d_2d_3\ldots$ differs from every $f(n)$ at position $n$.

Power set version:
Suppose $f:S\to\mathcal{P}(S)$. Let
$D=\{x:x\notin f(x)\}$
Then $D\notin\text{im}(f)$.
