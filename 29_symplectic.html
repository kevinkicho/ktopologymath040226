Action-Angle Variables (θ₁, θ₂)
In action-angle coordinates the dynamics is linear: θ̇ᵢ = ωᵢ = const. The torus [0,2π]² is the fundamental domain. The trajectory wraps as a straight line of slope ω₂/ω₁.
Arnold-Liouville Theorem: If an n-DOF Hamiltonian system has n independent conserved quantities in involution ({Hᵢ, Hⱼ} = 0), then it is completely integrable: phase space foliates into invariant n-tori parameterized by the action variables Iᵢ. Motion on each torus is quasi-periodic with frequencies ωᵢ = ∂H/∂Iᵢ. Rational ratios give closed (periodic) orbits; irrational ratios give dense (ergodic) orbits filling the torus.

Poisson Bracket

{f,g} = ∂f/∂q · ∂g/∂p − ∂f/∂p · ∂g/∂q

The Poisson bracket is the Lie bracket of Hamiltonian vector fields: X_{f,g} = [X_f, X_g]. Key identities:

{q,p} = 1 (canonical)

{H,f} = ḟ (equation of motion)

{f,g} = −{g,f} (antisymmetry)

{{f,g},h} + {{g,h},f} + {{h,f},g} = 0 (Jacobi)

Canonical Transformations

(q,p) → (Q,P) with dQ∧dP = dq∧dp

A transformation is canonical if it preserves the symplectic form ω. Equivalently its Jacobian matrix J satisfies JᵀΩJ = Ω where Ω is the standard symplectic matrix.

The left panel shows trajectories in (q,p) coordinates; the right panel shows the same trajectories in the new (Q,P) coordinates. Shapes deform but enclosed areas are preserved.

Four Generating Functions

F₁(q,Q): p=∂F₁/∂q, P=−∂F₁/∂Q

F₂(q,P): p=∂F₂/∂q, Q=∂F₂/∂P

F₃(p,Q): q=−∂F₃/∂p, P=−∂F₃/∂Q

F₄(p,P): q=−∂F₄/∂p, Q=∂F₄/∂P

Example: Q=q², P=p/2q is generated by F₂(q,P) = q²P, giving p=∂F₂/∂q = 2qP = 2q·(p/2q) = p ✓ and Q=∂F₂/∂P = q²✓.

Physical significance: Every canonical transformation corresponds to a symmetry or a change of representation. The action-angle transformation is the canonical transformation that linearizes integrable dynamics.

Action Variables

Iᵢ = (1/2π) ∮ pᵢ dqᵢ = Eᵢ/ωᵢ

For the 2D harmonic oscillator H = ω₁I₁ + ω₂I₂. The action Iᵢ = Eᵢ/ωᵢ is the area enclosed by the i-th orbit in the (qᵢ, pᵢ) sub-plane divided by 2π — a topological invariant.

Angle Variables

θ̇ᵢ = ∂H/∂Iᵢ = ωᵢ = const

In action-angle coordinates (I,θ) the Hamiltonian depends only on the actions: H = H(I). Dynamics is linear on the torus: θᵢ(t) = ωᵢt + θᵢ(0).

Rational vs Irrational Frequencies

ω₁/ω₂ ∈ ℚ: closed orbit (Lissajous figure) — the trajectory returns to its starting point after a finite time.

ω₁/ω₂ ∉ ℚ: dense orbit — fills the torus ergodically. The golden ratio φ=(1+√5)/2 is the "most irrational" — KAM tori with this frequency are the last to break.

KAM Theorem

H = H₀(I) + εH₁(q,I,t)

For small perturbations ε, most tori survive if the frequencies satisfy the Diophantine condition |k·ω| ≥ γ|k|⁻ᵛ. Resonant tori break first; the golden-ratio torus is the last survivor. KAM theory proves persistence of quasi-periodic motion in nearly-integrable systems.

Geometric Quantization

Quantization is not merely "put hats on classical quantities." Geometric quantization shows how quantum mechanics emerges from symplectic geometry in three stages:

1. Prequantization

L → phase space, curv(∇) = ω/ħ

Construct a complex line bundle L over phase space whose curvature 2-form equals ω/ħ. Sections ψ of L are prequantum wave functions. The Dirac quantization condition: ω/2πħ must have integer cohomology class — this is why ∮ p dq = nħ.

2. Polarization

Lagrangian submanifold: T*Q ⊂ T*M

A polarization selects half the phase space directions (a Lagrangian foliation). Kähler polarization gives holomorphic wave functions; real polarization (vertical = momentum directions) recovers position-space QM: ψ(q).

3. Bohr-Sommerfeld Condition

∮_γ p dq = nħ, n ∈ ℤ

Only orbits γ on the torus for which the action integral is an integer multiple of ħ contribute. This selects quantized tori. For the harmonic oscillator: I = E/ω = nħ → E_n = nħω (or (n+½)ħω with metaplectic correction).

The winding number of the prequantum connection around the torus is the quantum number n — quantum mechanics is topology.
