Abstract Index Notation

Penrose's abstract indices label tensor slots, not components. Upper = contravariant, lower = covariant. Dotted (primed) = conjugate spinor space.

κA ∈ S  (unprimed spinor space)
κ̄A' ∈ S̄  (primed = complex conjugate)

ε-Spinor (Symplectic Metric)

The metric spinor εAB is the invariant antisymmetric form used to raise/lower indices:

εAB = [[0, 1], [−1, 0]]
κA = εAB κB
→ κ0 = κ¹, κ1 = −κ⁰

Twistors from Spinors

A twistor Zα is a pair of spinors:

Zα = (ωA, πA')

where ωA ∈ S and πA' ∈ S̄ are related by the incidence relation:

ωA = i xAA' πA'

Here xAA' is the spacetime position encoded as a 2×2 Hermitian matrix.

A null ray in spacetime (a light ray) corresponds to a point in real projective twistor space PN ⊂ PT

Peeling Theorem

Along an outgoing null geodesic at large distance r, the Weyl scalars fall off as:

Ψ₀ ~ r⁻⁵  (ingoing transverse)
Ψ₁ ~ r⁻⁴  (ingoing longit.)
Ψ₂ ~ r⁻³  (Coulomb)
Ψ₃ ~ r⁻²  (outgoing longit.)
Ψ₄ ~ r⁻¹  (outgoing transverse — RADIATION)

At null infinity ℐ⁺, only Ψ₄ survives — it is the gravitational wave strain:

h₊ + ih× ∝ ∫∫Ψ₄ dt dt

Goldberg-Sachs Theorem

A vacuum spacetime admits a shear-free null geodesic congruence if and only if it is algebraically special (Petrov type II, III, D, N, or O).

Type D (Schwarzschild, Kerr): κ=σ=0 along both PNDs
Type N (grav. wave): κ=σ=ρ=0 along the 4-fold PND
Generic (Type I): no shear-free geodesic congruence
